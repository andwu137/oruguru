# Install:
## From Source
```bash
git clone --recurse-submodules git@github.com:andwu137/oruguru.git
cd oruguru
make BUILD_MODE=RELEASE
./build/oruguru
```

# Development:
```bash
git clone --recurse-submodules git@github.com:andwu137/oruguru.git
cd oruguru
# make BUILD_MODE=DEBUG
make
./build/oruguru
```
