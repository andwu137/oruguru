# Install:
## From Source
```bash
git clone --recurse-submodules git@github.com:andwu137/oruguru.git
cd oruguru
# build desktop: `make raylib_desktop`
make release
./build/oruguru
```

# Development:
```bash
git clone --recurse-submodules git@github.com:andwu137/oruguru.git
cd oruguru
# build desktop: `make raylib_desktop`
make dev
./build/oruguru
```
